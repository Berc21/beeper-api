endpoints
---------
post /auth/register - register
post /auth/login - login

get /users/username - get user profile (name, email, avatar)
put /users/me/ - update settings
put /users/me/avatar - update avatar
get /users/id/beeps - get user beeps

get /beeps/ - get all beeps, ordered by date
get /beeps/id/ - get single beep (text, likes, comment count)
get /beeps/id/comments  - get beep comments
post /beeps/id/comments - post beep comments
patch /beeps/id/likes - like a beep
delete /beeps/id/likes - unlike a liked beep

post /comments/id/likes - like a comment
delete /comments/id/likes - unlike a comment
